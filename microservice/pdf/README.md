# PDF Generator Microservice

Высокопроизводительный микросервис для генерации PDF из массива URL изображений. 
Оптимизирован для работы на серверах с минимальным объемом оперативной памяти (256-512 MB).

## Особенности архитектуры
- **Extremely Low RAM**: Изображения не загружаются в память. Загрузка из сети и выдача PDF работают потоково (chunks).
- **Zero-copy PDF**: JPEG и PNG встраиваются в PDF-контейнер без перекодировки (исходный вес).
- **Graceful WEBP Fallback**: Форматы вроде WEBP локально конвертируются последовательно, не вызывая скачков потребления памяти.

## Запуск в Docker

1. Сборка образа:
```bash
docker build -t pdf-generator .
```
