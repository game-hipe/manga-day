# API Документация — Манга-сервис

Этот документ служит справочником по доступным эндпоинтам API. Здесь описаны все маршруты, методы, параметры и их назначение.

---

## Эндпоинты

### 🔹 Получение манги

| Эндпоинт | Описание | Метод |
|--------|--------|--------|
| `GET /manga/sku/{sku}` | Получить мангу по артикулу (SKU) | `GET` |
| `GET /manga/url/{url}` | Получить мангу по URL (поддерживаются источники из `src/spider`) | `GET` |
| `GET /manga/{id}` | Получить мангу по внутреннему ID в БД | `GET` |

---

### 🔍 Поиск манги

Все эндпоинты поиска поддерживают пагинацию:

| Параметр | Описание | По умолчанию |
|--------|--------|-------------|
| `page` | Номер страницы | `1` |
| `per_page` | Количество элементов на странице | `30` |

| Эндпоинт | Описание | Метод | Обязательные параметры |
|--------|--------|--------|------------------------|
| `GET /pages` | Получить список манги (все, без фильтров) | `GET` | — |
| `GET /pages/genre` | Получить мангу по жанру | `GET` | `query` — ID жанра |
| `GET /pages/author` | Получить мангу по автору | `GET` | `query` — ID автора |
| `GET /pages/language` | Получить мангу по языку | `GET` | `query` — ID языка |
| `GET /pages/query` | Поиск манги по текстовому запросу | `GET` | `query` — строка (например, `sister`) |

---

### 🏷️ Тэги и справочные данные

| Эндпоинт | Описание | Метод | Параметры |
|--------|--------|--------|----------|
| `GET /genres` | Получить все жанры | `GET` | — |
| `GET /languages` | Получить все языки | `GET` | — |
| `GET /authors` | Получить список авторов (с пагинацией) | `GET` | `page`, `per_page` |

> ⚠️ Примечание: В оригинале указан `/author`, но логичнее использовать множественное число `/authors` для соответствия REST-нормам.

---

### ⚙️ Утилиты (Alpha)

| Эндпоинт | Описание | Метод | Требования |
|--------|--------|--------|------------|
| `POST /manga/add` | Добавить новую мангу в базу | `POST` | Тело запроса должно соответствовать схеме `MangaSchema` |

> 🔒 Данный функционал находится в стадии Alpha. Используйте с осторожностью.

---

## 📌 Примечания

- Все URL-ы чувствительны к регистру.
- Для корректной работы некоторых эндпоинтов требуется наличие данных в БД и настроенные спайдеры (`src/spider`).
- Пагинация применяется автоматически, если не указан `per_page=None`.
